INSTRUCCIONES PARA EL EQUIPO EN WINDOWS
========================================================================

REQUISITOS:
- Java 17 o superior
- Maven 3.6+
- IDE (IntelliJ IDEA o Eclipse)

PASOS RÁPIDOS:

1. Instalar Java 17 y Maven
   Ver: documentacion/WINDOWS_SETUP.md (sección "Instalación")

2. Configurar variables de entorno
   JAVA_HOME: C:\Program Files\Eclipse Adoptium\jdk-17.x.x
   PATH: Agregar %JAVA_HOME%\bin y Maven\bin

3. Clonar/copiar el proyecto
   cd C:\Users\TuUsuario\Projects\biblioteca-pap

4. Compilar
   scripts\compile-all.bat
   O: mvn clean compile

5. Ejecutar
   scripts\ejecutar-app.bat
   O: mvn exec:java

CONFIGURACIÓN IDE (IMPORTANTE):
- IntelliJ IDEA:
  Run → Edit Configurations → Working directory: $PROJECT_DIR$

- Eclipse:
  Run → Run Configurations → Arguments → Working directory: ${project_loc}
